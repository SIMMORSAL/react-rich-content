<h1 align="center">React Rich Content</h1>

### Install:

```bash
yarn add react-rich-content
```

### Usage:

```ts
import { RichContent } from "react-rich-content";
...
return <RichContent
          backgroundColor="black"
          textColor="white"
          data={[...]}
        />
```
